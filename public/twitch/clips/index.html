Documentación

Endpoint API

Para crear clips desde comandos de chat, usa el siguiente endpoint:

POST /api/clips/create?channel={canal}

Parámetros:

  • channel (opcional): Nombre del canal donde crear el clip

Ejemplo para Nightbot

Nota: Nightbot no soporta POST requests directamente. Para usar esta funcionalidad en Nightbot, necesitarás un servicio intermedio o usar la interfaz web.

Rate Limiting

Para mantener un servicio justo, este endpoint está limitado a 3 clips exitosos por período de 5 minutos. Las solicitudes que excedan este límite recibirán un error.

Notas

  • Requiere autenticación con scope clips:edit
  • Los clips tardan unos segundos en procesarse
  • Se recomienda un cooldown de 5-10 segundos en comandos de chat
  • Puedes iniciar sesión con una cuenta de bot para que los clips aparezcan bajo ese nombre
